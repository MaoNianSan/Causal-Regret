Project: exp1_controlled_validity
Package purpose: clean rerun-ready EXP1 source archive

Included:
- source code and entry points
- self-check and code-check gates
- output and manuscript-interface contracts
- documentation for fast, full, rebuild, and trace-audit modes

Excluded intentionally:
- completed fast/full outputs
- partial or historical run logs
- executed result notebooks
- caches and virtual environments

The package is not evidence of a completed numerical run. New outputs must be
created locally or on the execution environment using the documented commands.
